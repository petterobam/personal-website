# 财务规划方法论

> 先保障，再安全，再自由。跳步是最大的风险。

## 核心概念

### 财务自由三阶段

1. **财务保障**：拥有 6~12 个月的生活储备金，不被意外击倒
2. **财务安全**：靠投资利息覆盖日常支出，不再被迫工作
3. **财务自由**：靠利息实现梦想生活，本金永不动用

### 三阶段目标计算公式

**财务保障目标**：
```
财务保障目标 = 月度生活成本 × 12
```

**财务安全目标**：
```
财务安全目标 = 年度生活成本 ÷ 预期年化收益率
```

**财务自由目标**：
```
财务自由目标 = 梦想年度成本 ÷ 预期年化收益率
```

### 50/50 还债原则

- 将 50% 的收入用于还债
- 保留 50% 的收入用于生活开支
- 快速偿还高息债务（信用卡、消费贷）

### 40:40:20 投资原则

- **40%**：低风险投资（货币基金、定期存款）
- **40%**：中风险投资（指数基金、债券）
- **20%**：高风险投资（股票、基金）

### 预算管理和财务监控

**预算制定**：
- 收入预算：稳定收入 + 预期收入
- 支出预算：必要支出 + 可选支出
- 储蓄预算：收入 - 支出 = 储蓄

**财务监控**：
- 每日记录支出
- 每周总结财务状况
- 每月分析财务数据
- 每季度调整投资策略

## 投资策略

### 保守型模型

- 预期年化收益率：3.0%
- 适用人群：风险厌恶者
- 资产配置：60% 低风险，40% 中风险

### 平衡型模型（推荐）

- 预期年化收益率：3.8%
- 适用人群：风险中性者
- 资产配置：40% 低风险，40% 中风险，20% 高风险

### 进取型模型

- 预期年化收益率：6.25%
- 适用人群：风险偏好者
- 资产配置：20% 低风险，40% 中风险，40% 高风险

## 实践步骤

1. **明确财务目标**：确定财务保障、安全、自由目标
2. **分析当前财务状况**：收入、支出、资产、负债
3. **制定预算**：收入预算、支出预算、储蓄预算
4. **建立紧急储备金**：6~12 个月的生活储备金
5. **偿还高息债务**：使用 50/50 还债原则
6. **开始投资**：采用 40:40:20 投资原则
7. **定期监控**：每日、每周、每月、每季度监控
