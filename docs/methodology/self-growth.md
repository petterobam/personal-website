# 自我成长方法论

> 目标设定、责任承担、持续成长

## 核心概念

### 目标设定的四个维度

1. **梦想**：你想要的生活是什么样子？
2. **价值观**：什么对你最重要？
3. **目标**：具体的、可衡量的目标
4. **策略**：如何实现目标的行动计划

### 责任 = 权力

把责任推给谁，就给了谁权力。承担责任是获得权力的前提。

**实践方法**：
- 遇到问题先问："我能做什么？"
- 不要抱怨，而是寻找解决方案
- 主动承担责任，而不是等待指示

### 竹子理论

**毛竹的生长规律**：
- 前 4 年：只长 3 厘米（扎根期）
- 第 5 年：90 天长到 20 米（爆发期）

**启示**：
- 早期看似没有进展，其实是在扎根
- 积累是爆发的前提
- 不要急于求成，坚持长期主义

### 好运的组成

好运 = 储存资本 + 识别机会 + 果断行动

**储存资本**：
- 能力资本：学习新技能，提升专业能力
- 关系资本：建立人脉网络，维护人际关系
- 财务资本：储蓄和投资，积累财务基础

**识别机会**：
- 保持开放心态，关注行业动态
- 培养洞察力，识别潜在机会
- 勇于尝试，不要害怕失败

**果断行动**：
- 一旦识别机会，立即行动
- 快速试错，快速迭代
- 不要犹豫，机会稍纵即逝

### 金钱观改造

**错误的金钱观**：
- 金钱是万恶之源
- 谈钱俗气，谈理想高尚
- 富人都是坏人
- 我不配拥有财富

**正确的金钱观**：
- 金钱是工具，不是目的
- 金钱是价值的交换
- 财富是实现梦想的手段
- 我值得拥有财富

**改变方法**：
1. 识别自己的金钱观
2. 分析哪些信念阻碍你获得财富
3. 用新的信念替代旧信念
4. 用实践验证新信念

## 实践步骤

1. **明确梦想**：写下你想要的生活
2. **分析价值观**：识别什么对你最重要
3. **设定目标**：设定具体、可衡量的目标
4. **制定计划**：分解目标，制定行动计划
5. **承担责任**：主动承担责任，不抱怨
6. **持续行动**：坚持长期主义，持续学习
7. **创造运气**：储存资本，识别机会，果断行动
