# 收入增长方法论

> 整天工作的人没有时间赚钱。目标是构建系统，不是出卖时间。

## 核心概念

### 收入层次模型

1. **主动收入**：出卖时间获得收入（工资、兼职）
2. **半被动收入**：系统化后减少时间投入（副业、小生意）
3. **被动收入**：系统自动运行，无需时间投入（投资、 royalties）

### 收入翻倍路径

**三支柱模型**：
- **主业**：稳定的基础收入
- **副业**：额外的收入来源
- **投资**：资产增值和被动收入

**收入公式**：
```
总收入 = 主业收入 + 副业收入 + 投资收入
```

### 开源优先于节流

**开源**：
- 寻找额外的收入来源
- 提升技能，增加主业收入
- 开发副业，构建收入系统
- 投资理财，获得被动收入

**节流**：
- 减少不必要的支出
- 优化固定支出（房租、保险、订阅）
- 理性消费，避免冲动购买
- 预算管理，控制支出

**原则**：
- 开源是进攻，节流是防守
- 先开源，再节流
- 开源优先于节流

### 企业家收入路径

**企业家收入公式**：
```
企业家收入 = 能力 × 精力 × 影响力 × 创意 ÷ 2
```

**关键要素**：
- **能力**：专业技能和经验
- **精力**：时间和投入
- **影响力**：品牌和声誉（最重要的乘数）
- **创意**：创新和差异化

**三条企业家路径**：
1. **AIGC 技术咨询专家**：风险最低，收入稳定，能力匹配，可扩展
2. **AI 换脸产品创业**：风险中等，收入潜力大，需要产品化
3. **AI 内容生成工具**：风险最高，收入潜力最大，需要团队

## 实践步骤

1. **分析当前收入结构**：主业、副业、投资
2. **识别收入增长机会**：技能提升、副业开发、投资优化
3. **制定收入增长计划**：短期、中期、长期目标
4. **提升主业收入**：技能提升、职位晋升、跳槽
5. **开发副业收入**：咨询、产品、服务
6. **增加投资收入**：优化投资组合，提高收益率
7. **构建企业家路径**：能力 × 精力 × 影响力 × 创意
