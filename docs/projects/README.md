## 项目案例

### [项目 1：财富自由之路 - 个人财务管理软件](https://github.com/yourusername/wealth-freedom)

**技术栈**：Electron + Vue 3 + TypeScript + Element Plus + ECharts
**项目类型**：PC + 移动端个人财务管理软件
**项目状态**：已发布 v0.1.0
**链接**：[GitHub](https://github.com/yourusername/wealth-freedom) | [下载](https://github.com/yourusername/wealth-freedom/releases)

**简介**：
帮助普通人走向财务自由的个人财务管理软件。让财务自由不只是概念，而是每天可量化、可追踪、可行动的目标。

**核心功能**：
- 财务现状看板（净资产、收支、负债）
- 财务自由目标计算器（三阶段目标）
- 资产配置可视化工具
- 投资记录与收益追踪
- AI 财务顾问

**技术亮点**：
- Electron 跨平台打包（macOS + Windows）
- Vue 3 Composition API
- TypeScript 类型安全
- Element Plus 组件库
- ECharts 数据可视化
- 响应式设计

**收获**：
- 理解用户需求，设计易用的产品
- 掌握 Electron 跨平台开发
- 提升前端工程化能力
- 学习财务管理和投资知识

### [项目 2：资产配置可视化工具](https://github.com/yourusername/wealth-freedom/tree/main/apps/desktop)

**技术栈**：Vue 3 + TypeScript + Element Plus + ECharts
**项目类型**：单页面应用
**项目状态**：已完成，已集成到财富自由之路软件

**简介**：
帮助用户可视化不同风险配置下的资产组合和预期收益，支持交互式调整配置比例。

**核心功能**：
- 财务阶段选择器（保障、安全、自由）
- 资产配置饼图 + 交互式滑块
- 风险-收益雷达图
- 配置详情表格
- 收益预测曲线
- 导出 PDF 功能

**技术亮点**：
- 响应式图表布局
- 交互式数据绑定
- PDF 导出功能
- 优化的性能和用户体验

**收获**：
- 掌握 ECharts 高级用法
- 理解数据可视化设计原则
- 提升用户体验设计能力
- 学习资产配置理论
